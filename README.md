### Install

#### From Terminal
```bash
apm install ghar-syntax
```

#### From Settings
**Settings** > **Install** > **Select 'Themes'** > **Search 'ghar-syntax'**.

Activate by going to the __Settings > Themes__ section and selecting it from the __Syntax Themes__ drop-down menu.
